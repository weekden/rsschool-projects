# Description to start of projects

## Fun Chat

A mini chat application with simple registration for exchanging messages.  
Implemented using WebSocket protocol.
You have to clone the [server application](https://github.com/rolling-scopes-school/fun-chat-server/tree/main) to your local computer, launch the local server, and maintain the server application's operation as needed.

---

## Async Race
A small project simulating requests using a car racing example. Results are saved and sorted.
You have to clone the [server application](https://github.com/mikhama/async-race-ap) to your local computer, launch the local server, and maintain the server application's operation as needed.
